Desktop Files Helper Pro Privacy Policy

Effective Date: 2026-05-10

Desktop Files Helper Pro is a local desktop productivity app. We respect your privacy and data security. This policy explains how data is handled.

1. Data Collection

The app does not collect, upload, share, or sell personal data. There is no account system, sign-in, cloud sync, analytics tracking, or ad profiling by default.

2. Internet and Data Transmission

The app is designed to work offline. Core features do not require internet access, and the app does not proactively transmit user data to developer-operated servers.

3. Local Storage

To provide functionality, the app may store configuration data locally on the device (for example, shortcut settings, quick-file library settings, UI preferences, and library paths). This data remains on the user device and is not uploaded to developer servers.

4. System Access and Purpose

The app accesses local resources only when the user explicitly uses related features, such as:

  • Selecting local files and folders (shortcuts, quick files, organize, preview)
  • Copying or moving files via the clipboard and Windows File Explorer (user-initiated)
  • Executing local PowerShell, Batch, or CMD scripts created or confirmed by the user
  • Reading local configuration and operation logs (for example, user_files_log.md stored in the user’s library folder)

These operations are strictly for app functionality and not for identity tracking, profiling, or commercial analytics.

5. Third-Party Services

By default, the app does not include third-party advertising SDKs, analytics SDKs, or behavioral tracking services. If online features are introduced in the future, this policy will be updated accordingly.

6. Security

Because data is primarily stored locally, users should protect their devices and environments (account security, system updates, etc.).

7. Children's Privacy

The app is not specifically directed to children and does not intentionally collect children's personal information.

8. User Choices

Since no cloud personal data is collected, requests such as cloud data export, correction, or deletion generally do not apply. Users may remove local data at any time by deleting local configuration files or uninstalling the app.

9. Policy Updates

If data practices change, this policy will be updated with a revised effective date.

10. Contact

For privacy questions, contact the developer via the contact information shown on the Microsoft Store listing.
