# Desktop Files Helper Pro Privacy Policy Effective Date: 2026-05-10 **Store status:** Policy text is ready; **public URL not yet published** (see `07-GitHub发布隐私政策URL完整流程.md`). Desktop Files Helper Pro is a local desktop productivity app. We respect your privacy and data security. This policy explains how data is handled. 1. Data Collection The app does not collect, upload, share, or sell personal data. There is no account system, sign-in, cloud sync, analytics tracking, or ad profiling by default. 1. Internet and Data Transmission The app is designed to work offline. Core features do not require internet access, and the app does not proactively transmit user data to developer-operated servers. 1. Local Storage To provide functionality, the app may store configuration data locally on the device (for example, shortcut settings, command settings, and UI preferences). This data remains on the user device and is not uploaded to developer servers. 1. System Access and Purpose The app accesses local resources only when the user explicitly uses related features, such as: - selecting local file paths for shortcuts - executing local scripts/commands created or confirmed by the user - reading local config files to restore settings These operations are strictly for app functionality and not for identity tracking, profiling, or commercial analytics. 1. Third-Party Services By default, the app does not include third-party advertising SDKs, analytics SDKs, or behavioral tracking services. If online features are introduced in the future, this policy will be updated accordingly. 1. Security Because data is primarily stored locally, users should protect their devices and environments (account security, system updates, etc.). 1. Children's Privacy The app is not specifically directed to children and does not intentionally collect children's personal information. 1. User Choices Since no cloud personal data is collected, requests such as cloud data export/correction/deletion generally do not apply. Users may remove local data at any time by deleting local config files or uninstalling the app. 1. Policy Updates If data practices change, this policy will be updated with a revised effective date. 1. Contact For privacy questions, contact the developer via the contact information shown on the Microsoft Store listing.
